# Course Expansion Plan: Duolingo-Scale Mastery Courses

> Goal: Take each course from its current size to Duolingo-scale depth, so learners go from zero knowledge to genuine mastery.

---

## Duolingo's Actual Scale

Duolingo's Spanish course (their gold standard):
- **8-10 sections** (major topic areas)
- **~40 units per section** (sub-topic clusters of 3-7 lessons)
- **~200+ total units** across the course
- **~1,000+ total lessons**
- **~15,000+ questions**
- **Completion: 6-12 months** at 5-10 min/day

### How Their Structure Maps to Ours

| Duolingo | Octokeen equivalent | Size |
|----------|-------------------|------|
| Section | **Section** (new layer, groups units) | 8-12 per course |
| Unit | **Unit** (our current "unit") | 15-25 per section |
| Lesson | **Lesson** (our current "lesson") | 3-7 per unit |

### Our Targets Per Course

| Metric | Current avg | Target | Multiplier |
|--------|------------|--------|-----------|
| Sections | 1 (flat) | 8-10 | New layer |
| Units | 10-13 | 80-120 | 8-10x |
| Lessons | 80-168 | 500-800 | 5-6x |
| Questions | 650-1,350 | 5,000-8,000 | 5-8x |
| Question types | 8-17 | 14+ | All types used |
| Completion time | 3-6 weeks | 4-8 months | 5-8x |

## Structure Rules for Expanded Courses

### Section Arc (8-10 sections per course)

Every course follows this arc:

| Section | Phase | Purpose | Difficulty |
|---------|-------|---------|-----------|
| 1 | **Welcome** | Absolute basics. "What even is this?" | Trivial |
| 2 | **Foundations** | Core vocabulary and mental models | Easy |
| 3 | **Building Blocks** | Essential concepts and relationships | Easy-Medium |
| 4 | **Core Skills** | Apply knowledge to simple problems | Medium |
| 5 | **Intermediate** | Combine concepts, handle nuance | Medium |
| 6 | **Advanced Concepts** | Deep knowledge, edge cases | Medium-Hard |
| 7 | **Real World** | Professional application, case studies | Hard |
| 8 | **Expert Topics** | Specialized knowledge, advanced analysis | Hard |
| 9 | **Mastery** | Synthesis, teaching others, professional scenarios | Very Hard |
| 10 | **Capstone** | Final challenges, comprehensive review | Expert |

### Unit Structure (15-25 units per section)

Each unit is a sub-topic cluster. Example: Section "Investing" might have units like:
- What is investing?
- Compound interest
- Stocks basics
- How to read a stock
- Bonds basics
- Index funds
- ETFs vs mutual funds
- Risk and diversification
- Your first trade
- Dollar-cost averaging
- Market crashes
- Dividends
- Taxes on investments
- Investing mistakes
- Section review

Each unit has **3-7 lessons** following the existing lesson rules from the content writing guide.

### Pacing Rules

1. **No more than 3 new concepts per unit.** If a unit introduces more, split it.
2. **Every 5th unit is a review unit.** It mixes questions from the previous 4 units. This is how we build spaced repetition into the course itself.
3. **Every section ends with a checkpoint.** A harder test that the learner must pass to continue.
4. **Difficulty never jumps more than 1 level between consecutive units.**
